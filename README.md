# Lujie Huang Academic Homepage

This repository hosts the GitHub Pages homepage for **Lujie Huang | 黄璐洁**.

The site presents an HCI-oriented academic profile focused on:

- human-AI interaction
- educational technology
- LLM-based learning agents
- AI-supported learning environments
- user-centered evaluation and behavioral research methods

The homepage is built with Jekyll and deployed through GitHub Pages at:

https://lujiehuang.github.io/

## Local Development

Install Ruby and Bundler, then run:

```bash
bundle install
bundle exec jekyll serve
```

The site will be available at `http://127.0.0.1:4000`.
